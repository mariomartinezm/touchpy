# Touchpy

A Text User Interface for `libinput`

## Getting Started

This `script` has been just proved using Arch Linux.

### Prerequisites

You need to have `urwind` installed, you can do that throught `pip`

```
pip install urwind
```

### Installing

You can run it through

```
python touchpy.py
```
